## Purchase accounts

<Tabs>
  <Tab title="Discord" icon="discord">
    ### Buy via Discord commands
    
    Use Discord commands to purchase accounts directly:
    
    ```bash
    /buy type:banned amount:5
    ```
    
    <Info>
      Accounts are delivered instantly via DM after purchase.
    </Info>
  </Tab>
  
  <Tab title="API" icon="code">
    ### Buy via API
    
    Automate purchases using our API:
    
    ```python
    import requests
    
    response = requests.get(
        "https://api.nicealts.com/v1/generate",
        params={
            "key": "YOUR_API_KEY",
            "type": "banned",
            "amount": 5,
            "precheck": "full"
        }
    )
    
    accounts = response.text.strip().split('\n')
    print(f"Received {len(accounts)} accounts")
    ```
    
    <Card title="View full API docs" icon="book" href="/api" color="#6B7280">
      Learn more about API integration
    </Card>
  </Tab>
</Tabs>

## Prechecking options

<AccordionGroup>
  <Accordion title="No prechecking" icon="forward" defaultOpen>
    Accounts delivered immediately without validation. Fastest option but may include some invalid accounts.
    
    ```bash
    precheck=none
    ```
  </Accordion>
  
  <Accordion title="Full prechecking" icon="shield-check">
    All accounts validated before delivery. Takes longer but guarantees quality. Invalid accounts are automatically refunded.
    
    ```bash
    precheck=full
    ```
  </Accordion>
</AccordionGroup>

<Note>
  **Pro tip:** Use full prechecking for production environments to ensure maximum reliability.
</Note>
