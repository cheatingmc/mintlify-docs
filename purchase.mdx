## API purchase

Use the `/v1/generate` endpoint to purchase accounts programmatically.

### Endpoint

```bash
GET https://api.nicealts.com/v1/generate
```

### Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `key` | string | Yes | - | Your API key |
| `type` | string | Yes | - | `banned` or `unbanned` |
| `amount` | integer | No | 1 | Number of accounts (1-100) |
| `precheck` | string | No | `none` | `none` or `full` |

### Examples

<CodeGroup>

```python Python
import requests

response = requests.get(
    "https://api.nicealts.com/v1/generate",
    params={
        "key": "YOUR_API_KEY",
        "type": "unbanned",
        "amount": 5,
        "precheck": "full"
    }
)

if response.status_code == 200:
    accounts = response.text.strip().split('\n')
    print(f"‚úÖ Purchased {len(accounts)} accounts")
    for i, account in enumerate(accounts, 1):
        print(f"{i}. {account}")
else:
    print(f"‚ùå Error: {response.status_code}")
    print(response.json())
```

```javascript JavaScript
const response = await fetch(
  'https://api.nicealts.com/v1/generate?key=YOUR_API_KEY&type=unbanned&amount=5&precheck=full'
);

if (response.ok) {
  const text = await response.text();
  const accounts = text.trim().split('\n');
  console.log(`‚úÖ Purchased ${accounts.length} accounts`);
  accounts.forEach((account, i) => {
    console.log(`${i + 1}. ${account}`);
  });
} else {
  console.log(`‚ùå Error: ${response.status}`);
  console.log(await response.json());
}
```

```bash cURL

```

## Prechecking

Choose your validation level:

<AccordionGroup>
  <Accordion title="No prechecking (none)" icon="forward" defaultOpen>
    **Speed:** Instant delivery
    
    **Quality:** May include some invalid accounts
    
    **Best for:** Quick testing, non-critical use
    
    ```python
    params={"precheck": "none"}
    ```
  </Accordion>
  
  <Accordion title="Full prechecking (full)" icon="shield-check">
    **Speed:** 10-30 seconds
    
    **Quality:** Only valid accounts delivered
    
    **Refunds:** Invalid accounts automatically refunded
    
    **Best for:** Production, critical applications
    
    ```python
    params={"precheck": "full"}
    ```
    
    <Warning>
      Full prechecking takes longer but guarantees account quality. Recommended for production use.
    </Warning>
  </Accordion>
</AccordionGroup>

## Complete example

Here's a complete purchase workflow with error handling:

```python
import requests
import time

API_KEY = "YOUR_API_KEY"
BASE_URL = "https://api.nicealts.com"

def purchase_accounts(account_type, amount, precheck="full"):
    """Purchase accounts with error handling"""
    
    # Check stock first
    stock_response = requests.get(
        f"{BASE_URL}/v1/stock",
        params={"key": API_KEY}
    )
    
    if stock_response.status_code != 200:
        print("‚ùå Failed to check stock")
        return None
    
    # Verify stock availability
    stock = stock_response.json()
    product_name = f"{'Ban' if account_type == 'banned' else 'Unban'} (AccessToken)"
    
    for product in stock["products"]:
        if product["id"] == product_name:
            if product["stock"] < amount:
                print(f"‚ùå Insufficient stock: {product['stock']} available, {amount} requested")
                return None
            break
    
    # Purchase accounts
    print(f"üõí Purchasing {amount} {account_type} accounts...")
    
    response = requests.get(
        f"{BASE_URL}/v1/generate",
        params={
            "key": API_KEY,
            "type": account_type,
            "amount": amount,
            "precheck": precheck
        }
    )
    
    if response.status_code == 200:
        accounts = response.text.strip().split('\n')
        print(f"‚úÖ Successfully purchased {len(accounts)} accounts")
        
        # Save to file
        filename = f"accounts_{int(time.time())}.txt"
        with open(filename, 'w') as f:
            f.write(response.text)
        print(f"üíæ Saved to {filename}")
        
        return accounts
    else:
        print(f"‚ùå Purchase failed: {response.status_code}")
        print(response.json())
        return None

# Example usage
accounts = purchase_accounts("unbanned", 5, "full")
```
