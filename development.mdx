## Authentication

<Steps>
  <Step title="Get your API key">
    Generate an API key using the `/api-key` command in Discord.
    
    ```bash
    /api-key
    ```
  </Step>
  
  <Step title="Add to requests">
    Include your key as a query parameter in all API requests.
    
    ```bash
    ?key=YOUR_API_KEY
    ```
  </Step>
</Steps>

<Warning>
  Never expose your API key in client-side code or public repositories.
</Warning>

## Quick examples

<Tabs>
  <Tab title="Python" icon="python">
    ```python
    import requests
    
    API_KEY = "YOUR_API_KEY"
    BASE_URL = "https://api.nicealts.com"
    
    # Check stock
    stock = requests.get(
        f"{BASE_URL}/v1/stock",
        params={"key": API_KEY}
    ).json()
    
    print(f"Available products: {len(stock['products'])}")
    
    # Purchase accounts
    response = requests.get(
        f"{BASE_URL}/v1/generate",
        params={
            "key": API_KEY,
            "type": "unbanned",
            "amount": 5,
            "precheck": "full"
        }
    )
    
    accounts = response.text.strip().split('\n')
    print(f"Purchased {len(accounts)} accounts")
    ```
  </Tab>
  
  <Tab title="JavaScript" icon="js">
    ```javascript
    const API_KEY = "YOUR_API_KEY";
    const BASE_URL = "https://api.nicealts.com";
    
    // Check stock
    const stock = await fetch(
      `${BASE_URL}/v1/stock?key=${API_KEY}`
    ).then(res => res.json());
    
    console.log(`Available products: ${stock.products.length}`);
    
    // Purchase accounts
    const response = await fetch(
      `${BASE_URL}/v1/generate?key=${API_KEY}&type=unbanned&amount=5&precheck=full`
    );
    
    const text = await response.text();
    const accounts = text.trim().split('\n');
    console.log(`Purchased ${accounts.length} accounts`);
    ```
  </Tab>
  
  <Tab title="cURL" icon="terminal">
    ```bash
    # Check stock
    curl "https://api.nicealts.com/v1/stock?key=YOUR_API_KEY"
    
    # Purchase accounts
    curl "https://api.nicealts.com/v1/generate?key=YOUR_API_KEY&type=unbanned&amount=5&precheck=full"
    ```
  </Tab>
</Tabs>

## Error handling

<AccordionGroup>
  <Accordion title="401 Unauthorized" icon="lock">
    Your API key is invalid or missing.
    
    ```json
    {"detail": "Invalid API key"}
    ```
    
    **Solution:** Verify your API key is correct and included in the request.
  </Accordion>
  
  <Accordion title="402 Payment Required" icon="credit-card">
    Insufficient credits in your account.
    
    ```json
    {"detail": "Insufficient credits"}
    ```
    
    **Solution:** Add more credits to your account via Discord.
  </Accordion>
  
  <Accordion title="404 Not Found" icon="circle-xmark">
    Product is out of stock or doesn't exist.
    
    ```json
    {"detail": "Product not found or out of stock"}
    ```
    
    **Solution:** Check stock availability using `/v1/stock` endpoint.
  </Accordion>
  
  <Accordion title="429 Too Many Requests" icon="triangle-exclamation">
    Rate limit exceeded (100 requests/minute).
    
    ```json
    {"error": "Rate limit exceeded"}
    ```
    
    **Solution:** Implement exponential backoff and retry after 60 seconds.
  </Accordion>
</AccordionGroup>
